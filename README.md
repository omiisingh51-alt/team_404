# SafeStep — Smart Accessible Navigation

Real-time accessible navigation system using audio cues, hazard detection, and crowd-sourced accessibility data for people with visual or mobility impairments.

**Team 404** | Udbhav '26 Hackathon (UDB-M7VE)

## Tech Stack
- **Frontend:** React, Vite, TailwindCSS v4, Lucide Icons
- **Backend:** Node.js, Express, MongoDB, JWT Auth
- **Features:** Demo mode (works without backend)

## Quick Start

```bash
# Frontend
cd client
npm install
npm run dev

# Backend (optional — app runs in demo mode without it)
cd server
npm install
npm run dev
```

## Deploy to Netlify
```bash
cd client
npm run build
# Upload the dist/ folder to Netlify
```
